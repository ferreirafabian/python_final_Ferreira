 proyecto BiciBlog, un blog local creado con Python y Django.

Este proyecto permite:

Registrar los kilómetros recorridos y pedidos entregados.
Registrar la cantidad de dinero recaudado.
Ver estadísticas totales de tus recorridos y ganancias.
Crear, editar y eliminar tus propios posts.
Actualizar tu perfil, cambiar avatar y restablecer la contraseña.
La aplicación incluye:

Autenticación de usuarios (registro, login, logout).
Formulario de creación de posts con selector de fecha mediante calendario.
Visualización de estadísticas con totales de kilómetros, dinero y pedidos.
Navbar con navegación a inicio, búsqueda por fecha, registrarse y acerca de mí.
Este proyecto es ideal para llevar un control de entregas y recorridos de forma sencilla y organizada.
